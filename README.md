## Build and Run

### Build the app

```bash
$ pnpm build:mac
```

### Install the app

```bash
# Mount the DMG installer
$ open dist/shadcn-electron-app-1.0.0.dmg

# Drag "Audio Recorder.app" to Applications folder
# Then run from Applications or Spotlight
```

### Run the app in development

```bash
$ pnpm dev
```
